# AI Programming Lab 1
## Project Setup
This project demonstrates proper environment setup for AI development.

## Environment
- Python 3.9
- NumPy
- Pandas
- Matplotlib
- scikit-learn

## Setup Instructions
1. Clone repository
2. Create environment: conda create -n ai_programming python=3.9
3. Activate: conda activate ai_programming
4. Install requirements: pip install -r requirements.txt
